# 评价 / 问大家反馈判断标准

## 一、先定义样本范围

| 数据来源 | 标准样本量 | 用途 |
| --- | --- | --- |
| 问大家 | 同类型竞品TOP10，每个链接至少50条，合计≥300条 | 判断购买前顾虑 |
| 评价 | 同类型竞品TOP10，每个链接至少200条，合计≥1000条 | 判断使用后满意点/痛点 |
| 中差评 | 每个竞品尽量采集全部中差评，至少100条 | 判断产品硬伤 |
| 追评 | 每个竞品至少50条 | 判断长期使用问题 |
| SKU评价 | 主销SKU评价单独统计 | 判断哪个规格/颜色/材质有问题 |
| 自家售后/客服记录 | 近30天/90天 | 判断我们自身是否也存在同类问题 |

# 二、反馈类型判断标准总表

| 用户反馈 | 说明 | 明确判断标准 | 动作 |
| --- | --- | --- | --- |
| 高频问大家 | 购买前顾虑 | 同一问题在问大家中占比≥10%，或TOP10竞品中≥5个竞品都出现该问题；如果占比≥15%，判定为核心购买顾虑；如果占比≥20%，必须进入主图/详情页/客服话术 | 主图/详情页提前解释，客服话术标准化 |
| 高频差评 | 产品硬伤 | 同一差评问题在中差评中占比≥10%，判定为产品痛点；占比≥15%，判定为严重痛点；占比≥20%，判定为产品硬伤，优先产品升级 | 产品升级优先级最高，不能只靠话术解决 |
| 高频好评 | 用户认可点 | 同一好评点在好评中占比≥15%，判定为用户认可卖点；占比≥25%，判定为核心成交卖点；如果好评点同时出现在标题/主图/评价中，可放大为主卖点 | 放大成主图第一卖点、详情页核心证据、标题词 |
| 竞品没解决的问题 | 差异化机会 | 某问题在竞品问大家/差评中占比≥10%，但TOP50竞品中明确解决或表达的商品少于5个，判定为差异化机会 | 做功能升级、服务承诺、主图差异表达 |
| 用户反复问但竞品没讲清楚 | 视觉表达机会 | 同一问题在问大家中占比≥10%，但竞品主图/详情页前3屏未解释，或客服回答不统一，判定为表达缺口 | 主图、详情页前3屏重点表达，客服话术统一 |
| 高频追评问题 | 长期使用风险 | 追评中同一问题占比≥8%，且多为使用一段时间后出现，如掉色、变形、塌陷、开线 | 改工艺、材质、质检标准，详情页避免过度承诺 |
| 高频SKU问题 | 规格/颜色/款式风险 | 某SKU负反馈占该SKU评价≥10%，或明显高于其他SKU 1.5倍以上 | 优化SKU、下架问题规格、调整主推SKU |
| 高频售后问题 | 履约/服务风险 | 退换货、尺寸不符、发错货、物流破损等问题占售后原因≥10% | 优化包装、尺码说明、发货质检、售后话术 |
| 低频但高风险问题 | 安全/合规/信任风险 | 即使占比＜5%，但涉及安全、过敏、儿童、异味、掉色、虚假宣传、材质造假，也要单独标红 | 优先核查，不等到高频再处理 |

# 三、频率等级标准

| 占比 | 等级 | 处理方式 |
| --- | --- | --- |
| ≥20% | 极高频 | 必须进入产品升级或主图详情页核心表达 |
| 15%-20% | 高频 | 必须进入卖点/痛点优化方案 |
| 10%-15% | 中高频 | 作为重点优化项 |
| 5%-10% | 中频 | 观察，结合竞品和利润判断是否处理 |
| 3%-5% | 低频 | 仅记录，除非是高风险问题 |
| ＜3% | 弱信号 | 暂不作为主要决策依据 |

# 四、五类反馈的完整落地标准

## 1. 高频问大家：购买前顾虑

### 判断逻辑

问大家代表用户**下单前不确定、不放心、怕踩坑**的问题。

### 常见类型

| 问大家类型 | 用户真实顾虑 | 典型问题 |
| --- | --- | --- |
| 材质顾虑 | 怕材质不好 | “会不会有味道？”“是不是A类？” |
| 尺寸顾虑 | 怕买错尺寸 | “1.8米沙发能用吗？” |
| 功能顾虑 | 怕功能没效果 | “真的防滑吗？”“防水吗？” |
| 适配顾虑 | 怕场景不适合 | “宿舍能用吗？”“有宠物能用吗？” |
| 清洁顾虑 | 怕后期麻烦 | “能机洗吗？”“好打理吗？” |
| 安全顾虑 | 怕儿童/老人用不安全 | “宝宝能用吗？”“老人会不会滑？” |
| 质量顾虑 | 怕不耐用 | “会不会起球/掉毛/变形？” |
| 服务顾虑 | 怕退换麻烦 | “尺寸不合适能退吗？” |

### 明确判断标准

| 条件 | 结论 |
| --- | --- |
| 某问题在问大家中占比≥10% | 购买前顾虑成立 |
| 某问题在TOP10竞品中≥5个竞品都出现 | 行业共性顾虑 |
| 某问题占比≥15% | 核心购买顾虑 |
| 某问题占比≥20% | 必须进入主图/详情页前3屏 |
| 问大家高频，但竞品主图没讲 | 视觉表达机会 |
| 问大家高频，但产品本身不能解决 | 产品能力不足，不能只靠话术 |

### 动作标准

| 问题类型 | 落地动作 |
| --- | --- |
| 材质顾虑 | 主图放材质特写，详情页放检测/材质说明 |
| 尺寸顾虑 | 主图或详情页放尺寸适配图 |
| 功能顾虑 | 主图做功能演示图，例如防滑、防水测试 |
| 场景顾虑 | 主图做具体使用场景 |
| 安全顾虑 | 放A类、安全、无味、检测报告 |
| 清洁顾虑 | 详情页加入清洗方式和使用说明 |
| 服务顾虑 | 加退换承诺、客服标准话术 |

## 2. 高频差评：产品硬伤

### 判断逻辑

差评代表用户**买完后不满意**，优先级高于问大家。  
问大家可以靠表达降低顾虑，差评很多时候必须靠产品解决。

### 常见类型

| 差评类型 | 代表问题 | 处理优先级 |
| --- | --- | --- |
| 材质差 | 薄、硬、粗糙、掉毛、有味 | 高 |
| 功能失效 | 不防滑、不防水、不遮光、不显白 | 高 |
| 尺寸不准 | 偏大、偏小、不贴合 | 高 |
| 做工差 | 开线、掉色、变形、破损 | 高 |
| 色差大 | 实物和图片不符 | 中高 |
| 使用麻烦 | 难清洗、难安装、容易移位 | 中高 |
| 包装/物流 | 破损、压痕、漏发 | 中 |
| 服务问题 | 客服不解决、退换困难 | 中高 |

### 明确判断标准

| 条件 | 结论 |
| --- | --- |
| 某差评问题占中差评≥10% | 产品痛点成立 |
| 某差评问题占中差评≥15% | 严重痛点 |
| 某差评问题占中差评≥20% | 产品硬伤 |
| 同一问题在TOP10竞品中≥5个都出现 | 行业普遍痛点 |
| 同一问题在自家评价中也出现≥10% | 自家必须优先整改 |
| 差评问题与功能卖点直接冲突 | 高风险，例如主打防滑但差评说不防滑 |
| 差评集中在某SKU | SKU风险，不一定是全品问题 |

### 动作标准

| 差评类型 | 动作 |
| --- | --- |
| 产品硬伤 | 改材质、改结构、改工艺、改供应商 |
| 功能虚弱 | 重新测试功能，达不到就不要主打 |
| 尺寸问题 | 增加尺寸图、适配表、SKU说明 |
| 色差问题 | 主图修正、增加实拍图、减少滤镜 |
| 包装问题 | 改包装、加防护、出厂检查 |
| 服务问题 | 改客服话术、退换承诺、售后流程 |
| SKU问题 | 调整主推SKU，问题SKU降权或下架 |

### 关键原则

**差评占比≥15%的问题，不能只靠主图包装解决，必须进入产品升级清单。**

---

## 3. 高频好评：用户认可点

### 判断逻辑

好评代表用户真正认可的成交理由，可以反向提炼主卖点。

### 常见类型

| 好评类型 | 可提炼卖点 |
| --- | --- |
| 材质好 | 亲肤、柔软、厚实、无味 |
| 功能好 | 防滑、防水、遮光、显白、耐脏 |
| 颜值高 | 高级感、百搭、奶油风、设计感 |
| 尺寸合适 | 贴合、适配、不移位 |
| 使用方便 | 好清洗、好安装、省心 |
| 性价比高 | 便宜、划算、超值 |
| 服务好 | 发货快、客服专业、售后好 |

### 明确判断标准

| 条件 | 结论 |
| --- | --- |
| 某好评点占好评≥15% | 用户认可卖点 |
| 某好评点占好评≥25% | 核心成交卖点 |
| 好评点同时出现在评价、问大家、标题词中 | 可作为强卖点 |
| 好评点与竞品差评点相反 | 可作为差异化卖点 |
| 好评点高频但主图没表达 | 主图卖点升级机会 |
| 好评点高频但标题没词 | 标题优化机会 |

### 动作标准

| 好评点 | 落地动作 |
| --- | --- |
| “很防滑” | 主图第一卖点：坐下不移位/强力防滑 |
| “没有味道” | 主图/详情页：开箱无异味 |
| “很高级” | 主图强化风格氛围 |
| “宝宝能用” | 详情页强化安全、A类、检测 |
| “尺寸刚好” | 增加尺寸适配图 |
| “好清洗” | 做清洗演示图 |
| “性价比高” | 强化组合装/到手价 |

### 关键原则

**高频好评不是简单放在详情页，而是要升级为：标题词、主图第一卖点、详情页证据链。**

## 4. 竞品没解决的问题：差异化机会

### 判断逻辑

如果用户有高频痛点，但竞品没有解决，就是产品和表达差异化机会。

### 明确判断标准

| 条件 | 结论 |
| --- | --- |
| 某问题在竞品问大家/差评中占比≥10% | 用户痛点成立 |
| TOP50中明确解决该问题的商品少于5个 | 供给不足 |
| 竞品详情页前3屏没有解释 | 表达不足 |
| 竞品评价中反复抱怨 | 产品未解决 |
| 用户愿意为解决该问题付费 | 可做产品升级 |
| 我方供应链能解决 | 可立项开发 |
| 我方供应链不能解决 | 只能做表达或服务承诺，不作为产品卖点 |

### 差异化机会分级

| 机会等级 | 判断标准 | 动作 |
| --- | --- | --- |
| A级机会 | 痛点占比≥15%，竞品解决少于5个，我方能解决 | 优先立项开发 |
| B级机会 | 痛点占比10%-15%，竞品表达弱，我方能表达清楚 | 主图/详情页差异化 |
| C级机会 | 痛点存在，但供应链解决难 | 观察或做服务承诺 |
| 不做 | 痛点低频、解决成本高、用户不愿付费 | 不优先 |

### 动作标准

| 竞品没解决的问题 | 我方动作 |
| --- | --- |
| 竞品不防滑 | 开发防滑底，主图做防滑测试 |
| 竞品有味道 | 做无味材质，详情页放检测 |
| 竞品尺寸不全 | 增加尺寸SKU |
| 竞品不适合儿童 | 做A类安全款 |
| 竞品主图看不懂 | 做更清楚的功能/场景表达 |
| 竞品客服解释差 | 建立标准客服话术 |

---

## 5. 用户反复问但竞品没讲清楚：视觉表达机会

### 判断逻辑

这类问题不一定要改产品，可能产品已有能力，但商家没讲清楚，导致用户反复问。

### 明确判断标准

| 条件 | 结论 |
| --- | --- |
| 某问题在问大家中占比≥10% | 用户有明确疑问 |
| 竞品主图前5张没有解释 | 主图表达缺口 |
| 竞品详情页前3屏没有解释 | 详情页表达缺口 |
| 竞品客服回答不统一 | 话术缺口 |
| 评价中用户买后认可该点 | 说明产品能解决，但前置表达不足 |
| 问大家高频，但差评不高 | 优先判断为表达问题，而非产品问题 |

### 表达机会判断

| 用户反复问 | 应该怎么表达 |
| --- | --- |
| 能不能机洗？ | 主图/详情页放清洗方式 |
| 会不会滑？ | 主图做防滑演示 |
| 有没有味道？ | 主图放无味/检测/开箱说明 |
| 尺寸怎么选？ | 做尺寸选择图 |
| 适合什么沙发/人群？ | 做场景适配图 |
| 厚不厚？ | 做厚度对比图 |
| 会不会掉色？ | 做色牢度/洗后对比 |
| 适不适合宝宝？ | 做A类/安全证明 |

### 动作标准

| 表达位置 | 处理方式 |
| --- | --- |
| 主图第1张 | 放第一点击理由，只讲最核心疑问 |
| 主图第2-3张 | 放功能证明或场景证明 |
| 主图第4-5张 | 放尺寸、材质、细节 |
| 详情页前3屏 | 解决购买前最大顾虑 |
| 详情页中段 | 讲证据、对比、参数 |
| 客服话术 | 针对高频问题统一标准回答 |
| 评价运营 | 引导买家反馈核心卖点 |

# 五、最终判断优先级

当反馈很多时，按这个顺序处理：

| 优先级 | 问题类型 | 原因 |
| --- | --- | --- |
| 1 | 高频差评 ≥15% | 已经影响成交和口碑，必须先改 |
| 2 | 高频问大家 ≥15% | 影响下单决策，必须前置解释 |
| 3 | 竞品没解决且用户高频抱怨 | 差异化机会，可能形成爆点 |
| 4 | 高频好评 ≥25% | 可放大为主卖点 |
| 5 | 用户反复问但竞品没讲清楚 | 表达优化机会，见效快 |
| 6 | 低频但高风险问题 | 虽低频，但涉及安全/合规，要单独处理 |

# 七、最终可直接放进方法论的版本

| 用户反馈 | 说明 | 完整判断标准 | 动作 |
| --- | --- | --- | --- |
| 高频问大家 | 购买前顾虑 | 同一问题在问大家中占比≥10%即成立；≥15%为核心顾虑；≥20%必须进入主图/详情页前3屏；若TOP10竞品中≥5个都出现，说明是行业共性顾虑 | 主图/详情页提前解释，客服话术标准化 |
| 高频差评 | 产品硬伤 | 同一问题在中差评中占比≥10%为产品痛点；≥15%为严重痛点；≥20%为产品硬伤；如果该问题与主打卖点冲突，优先级上升为P0 | 产品升级优先级最高，改材质、结构、工艺、SKU |
| 高频好评 | 用户认可点 | 同一好评点占好评≥15%为认可卖点；≥25%为核心成交卖点；如果好评点同时出现在评价、标题、问大家中，可升级为主卖点 | 放大成主图第一卖点、详情页证据、标题词 |
| 竞品没解决的问题 | 差异化机会 | 问大家/差评中问题占比≥10%，但TOP50中明确解决该问题的商品少于5个，且我方供应链或服务能解决，即判定为差异化机会 | 做功能升级、服务承诺、差异化主图表达 |
| 用户反复问但竞品没讲清楚 | 视觉表达机会 | 同一问题在问大家中占比≥10%，但竞品主图前5张或详情页前3屏没有解释，且差评占比不高，优先判定为表达问题 | 主图、详情页重点表达，客服话术统一 |

---

一句话总结：

**问大家看“下单前顾虑”，差评看“产品硬伤”，好评看“成交卖点”，竞品没解决的问题看“差异化机会”，用户反复问但没人讲清楚的问题看“视觉表达机会”。**
