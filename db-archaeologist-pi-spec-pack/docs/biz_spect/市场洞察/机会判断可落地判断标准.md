# 机会判断可落地判断标准

把“四刀切机会”升级成一套**完整可落地判断标准**。核心不是“想到一个细分方向”，而是要判断：

**这个细分方向有没有需求、有没有增长、竞争强不强、我们能不能做、做了能不能赚钱。**

---

# 四刀切机会：完整可落地判断标准

## 一、四刀切机会总逻辑

| 四刀 | 核心问题 | 判断依据 | 最终输出 |
| --- | --- | --- | --- |
| 第一刀：细分人群 | 谁的需求没有被满足？ | 人群词、评价、问大家、达摩盘、竞品主图 | 人群定位 |
| 第二刀：细分产品 | 产品本身哪里能升级？ | 爆款基因、差评痛点、功能词、材质词、SKU结构 | 产品创新方案 |
| 第三刀：细分场景 | 用户在哪个具体场景使用？ | 场景关键词、主图场景、评价场景、内容平台场景 | 场景化产品/主图 |
| 第四刀：细分价格带 | 哪个价格带有机会？ | 价格带供给、销量、GMV、毛利、竞品强弱 | 定价与产品角色 |

最终判断公式：

```text
细分机会 = 需求存在 × 增长成立 × 竞争可切 × 我方可做 × 利润可承接

```
---

# 二、第一刀：细分人群

## 1. 判断目标

找到某一类用户的独特需求，然后形成专属产品定位。

不是简单写“学生党、宝妈、老人”，而是要判断：

1.  这个人群有没有搜索？
    
2.  这个人群有没有明确痛点？
    
3.  市场有没有专门产品承接？
    
4.  竞品有没有针对这个人群表达？
    
5.  我们能不能做出专属差异？
    

---

## 2. 看哪些数据

| 数据来源 | 看什么 |
| --- | --- |
| 关键词 | 是否有人群词，例如学生、老人、儿童、宝妈、男士、女生 |
| 搜索增长 | 人群词是否增长 |
| 问大家 | 用户是否反复问“适不适合某类人” |
| 评价 | 用户是否自报身份或使用对象 |
| 达摩盘 | 竞品购买人群是否集中 |
| 主图 | 竞品是否已经做人群表达 |
| 详情页 | 是否针对人群痛点解释 |
| 竞品数量 | 专门服务该人群的商品多不多 |

---

## 3. 明确判断标准

| 判断项 | 参考值 | 结论 |
| --- | --- | --- |
| 人群词在关键词TOP300中占比 | ≥8% | 明确人群需求成立 |
| 人群词近30天搜索增长 | ≥20% | 人群趋势成立 |
| 问大家中人群适配问题占比 | ≥10% | 购买前人群顾虑成立 |
| 评价中人群使用反馈占比 | ≥10% | 真实使用人群成立 |
| TOP50中明确做人群定位商品 | ＜5个 | 人群供给不足，有机会 |
| TOP50中明确做人群定位商品 | ≥15个 | 人群竞争较强，需要差异 |
| 达摩盘中某人群占比高于行业均值 | ≥20% | 可确认核心购买人群 |
| 竞品主图没人群表达，但关键词有人群需求 | 成立 | 主图人群化机会 |

---

## 4. 人群机会分级

| 等级 | 判断标准 | 动作 |
| --- | --- | --- |
| A级人群机会 | 人群词占比≥8%，增长≥20%，TOP50承接＜5个，我方能开发专属款 | 优先开发人群专属产品 |
| B级人群机会 | 人群词占比≥8%，但竞品已有承接，主图表达弱 | 做人群主图和详情页优化 |
| C级人群机会 | 人群词有增长，但搜索体量小 | 小批量测试 |
| D级人群机会 | 无搜索、无评价、无问大家证据 | 不单独做人群定位 |

---

## 5. 人群定位输出标准

| 人群 | 搜索证据 | 痛点证据 | 竞品缺口 | 定位 |
| --- | --- | --- | --- | --- |
| 学生党 | “学生宿舍”搜索增长25% | 怕贵、空间小、临时用 | TOP50只有3个明确表达宿舍 | 宿舍平价实用款 |
| 宝妈 | “儿童/A类/无味”占比12% | 怕不安全、有味道 | 竞品安全表达弱 | 宝宝可用安全款 |
| 老人 | “老人/防滑”增长22% | 怕摔、怕麻烦 | 防滑讲不清 | 老人防滑省心款 |
| 精致女生 | “高级感/显白/奶油风”增长 | 怕不好看、怕土 | 视觉同质化 | 高颜值风格款 |
| 轻熟龄女性 | “抗老/品质/通勤/气质” | 要体面、省心、品质 | 价格带可升级 | 品质气质款 |

## 6. 可落地动作

| 机会类型 | 产品动作 | 主图动作 | 标题动作 |
| --- | --- | --- | --- |
| 学生党 | 做低价、便携、小空间适配 | 宿舍场景 | 学生/宿舍/平价 |

| 轻熟龄女性 | 做品质、气质、耐用 | 轻熟/通勤/品质感 | 气质/品质/高级 |

# 三、第二刀：细分产品

## 1. 判断目标

找到产品本身可以升级、改造、组合、差异化的地方。

细分产品不是随便创新，而是从这些地方找：

1.  爆款款式能不能升级？
    
2.  竞品差评能不能解决？
    
3.  用户功能词是否增长？
    
4.  SKU是否有缺口？
    
5.  其他类目爆点能不能迁移？
    

---

## 2. 看哪些数据

| 数据来源 | 看什么 |
| --- | --- |
| 行业TOP50/TOP100 | 爆款产品形态、材质、功能 |
| 关键词 | 功能词、属性词、材质词是否高频 |
| 评价差评 | 产品硬伤在哪里 |
| 问大家 | 用户买前最担心什么 |
| 竞品SKU | 尺寸、颜色、组合是否不足 |
| 其他类目 | 是否有可借鉴的爆款元素 |
| 供应链 | 我方能不能实现升级 |

---

## 3. 细分产品判断标准

| 创新方式 | 明确判断标准 | 动作 |
| --- | --- | --- |
| 模仿 + 微创新 | 某产品形态TOP50占比≥30%，说明需求成立；但竞品差评中某痛点占比≥10% | 在成熟爆款基础上做材质、功能、视觉微升级 |
| 功能升级 | 功能词搜索占比≥10%，或问大家功能顾虑≥15%，或差评功能问题≥10% | 开发功能升级款 |
| 材质升级 | 某材质差评占比≥10%，或高价竞品因材质好销量高 | 换材质、加厚、做安全/舒适升级 |
| 组合创新 | 用户问大家中组合/套装/搭配需求≥10%，或多件购买明显 | 做组合装、家庭装、套装 |
| 打对手缺点 | 竞品同一差评问题≥10%，TOP50明确解决商品＜5个 | 直接把竞品缺点变成我方卖点 |
| 改款升级 | 同类形态成熟，但增长放缓；新形态高增速商品占比≥30% | 标准款改异形款、便携款、套装款 |
| 增加卖点 | 好评中某认可点≥15%，但竞品主图未表达 | 把好评点升级成主卖点 |
| 跨类目借鉴 | 其他相关类目TOP50中某元素出现≥10个，本类目TOP50＜5个 | 做跨类目创新款 |

---

## 4. 产品机会分级

| 等级 | 判断标准 | 动作 |
| --- | --- | --- |
| A级产品机会 | 需求已验证，痛点占比≥10%，我方能解决，毛利可承接 | 立项开发 |
| B级产品机会 | 有需求，有趋势，但供应链需验证 | 打样测试 |
| C级产品机会 | 只有内容热度，成交未验证 | 小批量观察 |
| D级产品机会 | 解决成本高，用户不愿付费 | 不开发 |

---

## 5. 产品创新输出模板

| 产品机会 | 证据 | 创新方式 | 我方产品动作 |
| --- | --- | --- | --- |
| 防滑升级款 | 竞品差评“容易滑”占比18% | 打对手缺点 | 升级防滑底，主图做防滑测试 |
| A类安全款 | 问大家“宝宝能用吗”占比15% | 功能/品质升级 | 做A类检测、安全无味款 |
| 组合装 | 问大家“能不能一套买”占比11% | 组合创新 | 做2件套/3件套/家庭装 |
| 异形款 | 异形款高增速商品占比35% | 改款升级 | 标准款延展异形款 |
| 奶油风款 | 小红书高赞，本类目供给少 | 跨类目/风格创新 | 做奶油风系列款 |

# 四、第三刀：细分场景

## 1. 判断目标

同一个产品，换一个使用场景，就可能变成一个新机会。

场景不是“放在哪里用”这么简单，而是要判断：

1.  用户是否按场景搜索？
    
2.  用户是否在场景中有特殊痛点？
    
3.  竞品是否做了场景化表达？
    
4.  这个场景能否带来新的产品形态、主图和标题？
    

---

## 2. 看哪些数据

| 数据来源 | 看什么 |
| --- | --- |
| 关键词 | 宿舍、客厅、军训、通勤、露营、办公室等场景词 |
| 主图 | 竞品是否表达场景 |
| 详情页 | 是否有场景解决方案 |
| 评价 | 用户真实使用场景 |
| 问大家 | 用户是否问“这个场景能不能用” |
| 小红书/抖音 | 场景内容是否火 |
| SKU | 是否有场景专属规格 |

---

## 3. 细分场景判断标准

| 判断项 | 参考值 | 结论 |
| --- | --- | --- |
| 场景词在关键词TOP300中占比 | ≥8% | 重要场景需求 |
| 场景词近30天搜索增长 | ≥20% | 趋势场景 |
| 评价中场景反馈占比 | ≥10% | 真实使用场景成立 |
| 问大家中场景适配问题占比 | ≥10% | 场景购买顾虑成立 |
| TOP50主图明确表达该场景 | ≥12个 | 主流场景 |
| TOP50主图明确表达该场景 | ＜5个 | 场景表达机会 |
| 小红书/抖音场景内容高热 | 多篇高赞/热视频 | 内容场景趋势 |
| 该场景需要特殊规格/功能 | 成立 | 可开发场景款 |

---

## 4. 场景机会分级

| 等级 | 判断标准 | 动作 |
| --- | --- | --- |
| A级场景机会 | 场景词占比≥8%，增长≥20%，TOP50表达＜5个 | 优先做场景款/场景主图 |
| B级场景机会 | 场景词稳定，竞品表达弱 | 优化主图和标题 |
| C级场景机会 | 内容平台热，但淘宝搜索弱 | 小批量测试 |
| D级场景机会 | 只是想象场景，无搜索无反馈 | 不做独立定位 |

---

## 5. 场景细分落地动作

| 场景动作 | 示例 | 落地方式 |
| --- | --- | --- |
| 同一产品换场景 | 家用、宿舍、办公室、户外、母婴 | 重新设计主图场景和标题词 |
| 同一功能换人群 | 防滑给老人，防水给母婴，易清洗给懒人 | 功能卖点不变，表达对象变 |
| 同一材质换表达 | 冰丝=夏季凉感，羊羔绒=冬季保暖 | 材质卖点场景化 |
| 同一产品换使用任务 | 通勤、军训、露营、收纳、送礼 | 用任务场景定义产品 |

---

## 6. 场景输出模板

| 场景 | 搜索证据 | 痛点证据 | 竞品表达 | 我方机会 | 动作 |
| --- | --- | --- | --- | --- | --- |
| 宿舍 | 场景词占比9%，增长22% | 怕贵、空间小 | TOP50仅4个明确表达 | 场景供给不足 | 做宿舍版主图和低价SKU |
| 军训 | 搜索增长30% | 怕晒黑、怕热 | 主图表达少 | 季节场景机会 | 做军训防晒神器 |
| 母婴 | 问大家占比15% | 怕不安全、有味 | 详情页解释弱 | 安全场景机会 | 做A类安全款 |
| 老人 | 防滑相关问题多 | 怕摔、怕麻烦 | 功能证明弱 | 功能场景机会 | 做老人防滑款 |

# 五、第四刀：细分价格带

## 1. 判断目标

找到哪个价格带可以切入，而不是盲目跟低价或高价。

价格带切分要回答：

1.  哪个价格带有流量？
    
2.  哪个价格带有GMV？
    
3.  哪个价格带竞争弱？
    
4.  哪个价格带毛利能做？
    
5.  哪个价格带适合我们的产品角色？
    

---

## 2. 看哪些数据

| 数据来源 | 看什么 |
| --- | --- |
| 行业TOP300价格 | 商品数量分布 |
| 支付买家数 | 成交集中在哪个价格带 |
| GMV | 金额集中在哪个价格带 |
| 毛利率 | 我方能不能赚钱 |
| 竞品数量 | 竞争强弱 |
| 小店/C店占比 | 是否普通商家有机会 |
| 搜索词 | 是否有价格意图词 |
| 评价 | 用户是否关注性价比/品质 |

---

## 3. 细分价格带判断标准

| 价格带机会 | 明确判断标准 | 动作 |
| --- | --- | --- |
| 高价市场做中价 | 中高价/高价商品数量占比≥60%，GMV占比≥60%，低价商品＜15% | 做高性价比切入，比主流高价低15%-30% |
| 低价市场做高价 | 低价/中低价商品数量占比≥60%，但差评中品质问题≥10%，高价供给＜15% | 做材质/功能/风格升级款 |
| 中间价格带空白 | 中价带商品数量占比＜15%，但支付买家数占比≥20%或搜索需求≥10% | 做大众主推款 |
| 同款多价格带 | 同一产品有低中高价需求，且各价格带支付买家数均≥15% | 做基础款、升级款、豪华款 |
| 低价销量高毛利低 | 支付买家数占比≥30%，但GMV占比低于销量占比 | 做引流款 |
| 高价GMV高竞品弱 | GMV占比≥30%，小店/C店占比≥30%，竞品痛点明显 | 做利润款或主推款 |
| 某价格带供给多成交少 | 商品数占比≥30%，支付买家数占比＜20% | 谨慎进入 |
| 某价格带需求低增长低 | 搜索低、成交低、GMV低 | 不进入 |

---

## 4. 价格带机会分级

| 等级 | 判断标准 | 动作 |
| --- | --- | --- |
| A级价格机会 | GMV占比≥30%，竞品弱，毛利可承接 | 开发主推/利润款 |
| B级价格机会 | 商品少，支付买家数≥20%，搜索有增长 | 小批量测试 |
| C级价格机会 | 有搜索，但成交弱 | 观察 |
| D级价格机会 | 商品多，销量低，毛利低 | 不进入 |

---

## 5. 价格带落地动作

| 价格策略 | 产品动作 | 主图动作 | 运营动作 |
| --- | --- | --- | --- |
| 高价市场做中价 | 保留核心功能，减少非必要成本 | 同功能更划算 | 抢高价市场的价格敏感用户 |
| 低价市场做高价 | 材质/功能/安全/设计升级 | 证明为什么贵 | 做品质利润款 |
| 中间价格带空白 | 做大众款 | 性价比+品质平衡 | 做主推爆款 |
| 同款多价格带 | 基础/升级/豪华 | 不同SKU不同表达 | 提升客单价 |
| 低价引流 | 极致控成本 | 价格利益清晰 | 拉访客和权重 |
| 高价利润 | 强价值证明 | 材质、设计、背书 | 提升毛利 |

# 六、四刀机会综合评分表

每个机会都要打分，不能只凭感觉。

| 评分维度 | 分值 | 判断标准 |
| --- | --- | --- |
| 需求强度 | 20分 | 搜索占比、关键词数量、问大家、评价是否支持 |
| 增长趋势 | 20分 | 搜索增长、商品增长、跨平台热度 |
| 竞争强度 | 15分 | TOP50承接数量、竞品强弱、头部集中度 |
| 产品可行性 | 15分 | 供应链、材质、功能、SKU能否实现 |
| 利润空间 | 15分 | 价格带、毛利、客单价 |
| 表达可行性 | 10分 | 主图、标题、详情页能否讲清 |
| 风险成本 | 5分 | 售后、合规、库存、季节风险 |

## 评分结论

| 总分 | 机会等级 | 动作 |
| --- | --- | --- |
| ≥85分 | A级机会 | 优先立项开发 |
| 70-84分 | B级机会 | 小批量测试 |
| 60-69分 | C级机会 | 继续观察或补数据 |
| ＜60分 | D级机会 | 暂不进入 |

# 七、四刀切机会最终输出表

| 切法 | 机会方向 | 判断标准 | 输出结果 |
| --- | --- | --- | --- |
| 细分人群 | 学生党、宝妈、老人、精致女生、轻熟龄女性 | 人群词占比≥8%，增长≥20%，竞品承接少 | 人群定位 |
| 细分产品 | 功能升级、材质升级、组合创新、打竞品缺点 | 差评/问大家≥10%，我方能解决 | 产品开发方案 |
| 细分场景 | 宿舍、军训、客厅、办公室、母婴、露营 | 场景词占比≥8%，TOP50表达少 | 场景化主图/产品 |
| 细分价格带 | 高价做中价、低价做高价、中间空白、多价格带 | 商品数、买家数、GMV、毛利交叉判断 | 定价和产品角色 |

# 八、最终总结

四刀切机会不是简单列出“人群、产品、场景、价格带”，而是每一刀都要有数据判断：

## 第一刀：细分人群

看：

**人群词搜索、人群痛点、竞品是否承接、达摩盘画像。**

判断：

**有没有一类人正在搜、正在问、正在买，但市场没有专门服务好。**

---

## 第二刀：细分产品

看：

**爆款共性、功能词、材质词、差评痛点、SKU缺口。**

判断：

**产品本身能不能通过微创新、升级、组合、改款解决用户痛点。**

---

## 第三刀：细分场景

看：

**场景词搜索、主图场景、评价场景、内容平台场景。**

判断：

**同一个产品能不能换一个更具体的使用场景重新卖。**

---

## 第四刀：细分价格带

看：

**商品数量、支付买家数、GMV、毛利、竞品强弱。**

判断：

**哪个价格带有需求、有成交、有利润、竞争还没那么强。**

一句话：

**人群决定卖给谁，产品决定凭什么买，场景决定什么时候用，价格带决定怎么成交。**
