# 具体统计方法

## 第一步：给每个商品打“主导人群标签”

每个商品最多打 2 个标签：

| 标签类型 | 说明 |
| --- | --- |
| 主人群标签 | 这个商品最主要服务谁 |
| 辅助人群标签 | 这个商品还顺带打动谁 |

例如：

| 商品表现 | 主人群 | 辅助人群 |
| --- | --- | --- |
| 低价、宿舍、便宜、实用 | 价格人群 | 场景人群 |
| 防滑、防水、易清洗 | 功能人群 | 品质信任人群 |
| 奶油风、高级感、客厅搭配 | 审美人群 | 场景人群 |
| A类、无味、儿童可用 | 品质信任人群 | 场景人群 |
| 礼盒、品牌、送长辈 | 身份品牌人群 | 品质信任人群 |
| 可爱、治愈、少女心 | 情绪悦己人群 | 审美人群 |

---

## 第二步：统计每类人群占比

### 1. 商品数量占比

```text
某人群标签商品数 ÷ 行业前300商品总数

```

用于判断这个人群是不是行业主流供给方向。

---

### 2. 销量占比

```text
某人群标签商品支付买家数 ÷ 前300总支付买家数

```

用于判断这个人群是否真实成交。

---

### 3. GMV占比

```text
某人群标签商品GMV ÷ 前300总GMV

```

用于判断这个人群是否有金额价值和利润空间。

---

### 4. 增长占比

```text
某人群标签中高增速商品数 ÷ 该人群商品总数

```

用于判断这个人群是不是趋势机会。

# 8大人群机会判断标准

| 数据表现 | 结论 | 动作 |
| --- | --- | --- |
| 数量占比高 + 销量占比高 | 主流成熟人群 | 可以做主推爆款，但要找差异 |
| 数量占比低 + 销量占比高 | 高效率机会人群 | 优先开发，可能供给不足 |
| 数量占比高 + 销量占比低 | 供给过剩人群 | 谨慎进入，除非有明显优势 |
| GMV占比高 + 销量占比低 | 高客单利润人群 | 适合做利润款 |
| 增长占比高 + 商品数少 | 趋势人群机会 | 快速测试新品 |
| 评价痛点集中 + 竞品未解决 | 产品升级机会 | 做功能/品质/服务升级 |
| 搜索词增长 + 主图表达少 | 视觉机会 | 做人群化主图 |
| 小店也能跑起来 | 低壁垒机会 | 快速跟进测试 |

# 最终输出结论模板

以后分析价格带和人群，不要这样写：

> 高价卖品质人群，中价卖大众人群，低价卖价格人群。

要这样写：

> 行业前300商品按8大人群统计后，价格人群占比最高，商品数量占比32%，但GMV占比只有21%，说明该类目低价供给多、成交多，但利润空间一般，适合作为引流款。  
审美人群商品数量占比18%，但GMV占比达到27%，说明用户愿意为风格和高级感付费，适合布局中高价风格款。  
品质信任人群商品数量占比仅9%，但高增速商品占比达到35%，说明安全、无味、A类材质正在成为趋势机会，可以优先开发品质升级款。

一句话：

**价格带不是单独判断，必须和8大人群交叉统计。最终要看：哪个人群有需求、哪个人群有增长、哪个人群有利润、哪个人群竞争还不强。**
