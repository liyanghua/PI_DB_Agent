# 关键词分析后的 8 个必输出结论与判断标准

核心原则：

**关键词结论不能只看搜索人气，要同时看：搜索体量、增长速度、竞争强度、转化意图、产品承接、利润空间。**

---

# 关键词分析后的 8 个必输出结论与判断标准

## 1. 当前类目最大需求是什么？

### 判断目标

判断这个类目里，用户最集中、最主流、最稳定的需求是什么。

### 核心判断字段

| 字段 | 作用 |
| --- | --- |
| 搜索人气 | 判断需求体量 |
| 搜索占比 | 判断该需求在类目中的份额 |
| 关键词数量 | 判断需求覆盖广度 |
| 支付转化率 / 点击转化率 | 判断需求是否能成交 |
| 相关商品数 | 判断供给规模 |
| TOP商品标题/主图是否承接 | 判断是否为行业主流需求 |

### 明确判断标准

| 判断标准 | 说明 |
| --- | --- |
| 某需求类型搜索人气占比 ≥ 25% | 可判断为当前类目最大需求 |
| 某需求类型关键词数量占比 ≥ 25% | 说明该需求覆盖广，用户搜索丰富 |
| TOP20关键词中，该需求词出现 ≥ 5个 | 说明头部流量集中在该需求 |
| TOP50商品中，承接该需求的商品占比 ≥ 30% | 说明行业主流商品也在围绕该需求做 |
| 搜索人气占比第一，且转化率不低于类目均值 | 可判定为“主流成交需求” |

### 结论模板

> 当前类目最大需求是【功能需求/场景需求/风格需求/品类需求】。判断依据是：该类需求搜索人气占比达到【X%】，关键词数量占比达到【X%】，TOP20关键词中出现【X】个，且TOP50商品中有【X%】商品在标题或主图中承接该需求，说明该需求既有用户搜索，也有商品成交承接。

## 2. 增长最快的需求是什么？

### 判断目标

判断哪些需求正在变热，适合作为新品测试、趋势款、内容种草方向。

### 核心判断字段

| 字段 | 作用 |
| --- | --- |
| 近7天搜索增长率 | 判断短期趋势 |
| 近30天搜索增长率 | 判断中期趋势 |
| 环比增长率 | 判断是否持续上升 |
| 新增关键词数量 | 判断是否出现新需求 |
| 飙升词数量 | 判断趋势密度 |
| 高增速商品数量 | 判断需求是否带动产品增长 |

### 明确判断标准

| 判断标准 | 说明 |
| --- | --- |
| 某需求词近7天搜索增长 ≥ 20% | 短期增长需求 |
| 某需求词近30天搜索增长 ≥ 30% | 中期增长需求 |
| 某需求类型下，有 ≥ 3个关键词同时增长 ≥ 20% | 可判断为需求趋势，而不是单个词偶然波动 |
| 该需求在高增速商品中占比 ≥ 30% | 说明需求已经带动商品增长 |
| 搜索增长 ≥ 20%，但商品承接少于TOP50中的5个 | 可判断为“趋势空白机会” |
| 搜索增长高，但成交/商品增长不同步 | 只能判定为“观察型趋势”，不能直接立项 |

### 增长需求分级

| 等级 | 标准 | 动作 |
| --- | --- | --- |
| 强趋势需求 | 近30天搜索增长 ≥ 30%，且高增速商品占比 ≥ 30% | 优先测款 |
| 潜力趋势需求 | 搜索增长 ≥ 20%，但商品承接少 | 快速小批量测试 |
| 伪趋势需求 | 搜索增长高，但无成交、无商品增长 | 观察，不立刻开发 |
| 活动型需求 | 短期突然增长，节后下降 | 只做短期活动款 |

### 结论模板

> 增长最快的需求是【场景/功能/风格/人群】需求。判断依据是：该需求下共有【X】个关键词近30天增长超过【X%】，其中【关键词A】【关键词B】增长最快；同时高增速商品中有【X%】商品承接该需求，说明该需求不是单个关键词波动，而是正在形成趋势。

## 3. 哪些需求我们已有产品承接？

### 判断目标

判断现有产品是否已经覆盖用户需求，避免重复开发，也判断哪些需求可以通过标题、主图、详情页优化快速承接。

### 核心判断字段

| 字段 | 作用 |
| --- | --- |
| 关键词需求词根 | 用户需求 |
| 自家商品标题 | 是否包含需求词 |
| 自家主图文案 | 是否表达需求 |
| 自家详情页卖点 | 是否解释需求 |
| 自家SKU | 是否有对应产品规格 |
| 自家评价 | 是否有用户反馈该需求 |
| 自家成交词 | 是否已有成交来源 |

### 明确判断标准

| 承接状态 | 判断标准 |
| --- | --- |
| 强承接 | 标题包含需求词 + 主图表达需求 + 详情页解释需求 + 有对应SKU + 有成交/评价反馈 |
| 中承接 | 标题包含需求词 + 产品具备对应功能/属性，但主图或详情页表达弱 |
| 弱承接 | 产品可能具备，但标题、主图、详情页都没有明确表达 |
| 未承接 | 没有对应产品、没有SKU、没有标题词、没有主图表达 |

### 可执行判断表

| 需求 | 标题是否承接 | 主图是否承接 | 详情页是否承接 | SKU是否承接 | 成交是否承接 | 判断 |
| --- | --- | --- | --- | --- | --- | --- |
| 防滑 | 是 | 是 | 是 | 是 | 是 | 强承接 |
| 宿舍 | 是 | 否 | 否 | 是 | 不明确 | 中承接 |
| 奶油风 | 否 | 否 | 否 | 否 | 否 | 未承接 |

### 结论模板

> 我们已有产品强承接的需求是【A/B/C】，因为对应商品在标题、主图、详情页、SKU和成交反馈中都有明确证据。  
中承接需求是【D/E】，产品本身具备，但表达不充分，建议优先优化标题和主图。  
弱承接或未承接需求是【F/G】，如果搜索体量和增长数据成立，可以进入新品开发或产品升级池。

## 4. 哪些需求我们没有产品承接？

### 判断目标

找出产品空白、SKU空白、视觉表达空白、标题承接空白。

### 核心判断字段

| 字段 | 作用 |
| --- | --- |
| 高搜索需求词 | 判断用户是否有需求 |
| 增长词 | 判断是否趋势机会 |
| 自家SKU | 判断有没有产品 |
| 自家标题 | 判断有没有搜索承接 |
| 自家主图 | 判断有没有点击承接 |
| 自家详情页 | 判断有没有转化承接 |
| 竞品承接情况 | 判断对手是否已经做了 |

### 明确判断标准

| 未承接类型 | 判断标准 | 动作 |
| --- | --- | --- |
| 产品未承接 | 有需求词，但自家没有对应产品/SKU | 新品开发 |
| 标题未承接 | 有产品，但标题没有对应关键词 | 标题优化 |
| 主图未承接 | 有产品，但主图没有表达需求 | 主图优化 |
| 详情页未承接 | 有产品，但详情页没有证明卖点 | 详情页优化 |
| 人群未承接 | 有产品，但没有针对具体人群表达 | 做人群版主图/详情 |
| 场景未承接 | 有产品，但没有场景化表达 | 做场景款或场景主图 |
| 价格带未承接 | 用户需求存在，但自家没有对应价格款 | 补SKU/补价格带 |

### 机会判断标准

| 数据表现 | 机会等级 |
| --- | --- |
| 搜索人气占比 ≥ 10%，自家无产品承接 | 高优先级机会 |
| 搜索增长 ≥ 20%，自家无产品承接 | 趋势机会 |
| 竞品少于TOP50中的5个，自家无承接 | 空白机会 |
| 竞品已大量承接，自家无承接 | 补课型机会 |
| 搜索低、增长低、自家无承接 | 暂不处理 |

### 结论模板

> 我们未承接的高优先级需求是【A/B/C】。判断依据是：这些需求搜索人气占比达到【X%】，近30天搜索增长达到【X%】，但自家商品没有对应SKU/标题/主图承接，因此属于产品或表达空白。建议优先进入新品开发或现有链接改版。

## 5. 哪些词适合做大爆款？

### 判断目标

判断哪些关键词适合支撑大流量、大成交、主推爆款链接。

### 大爆款关键词标准

大爆款词必须同时满足：

| 维度 | 明确标准 |
| --- | --- |
| 搜索体量 | 搜索人气进入类目TOP20，或搜索人气占比 ≥ 5% |
| 需求稳定性 | 近30天搜索波动不超过 ±20%，不是短期异常词 |
| 成交能力 | 支付转化率不低于类目均值，最好高于均值 |
| 商品承接 | TOP50商品中有 ≥ 10个商品承接该词 |
| 人群广度 | 不是极窄人群，适合较大范围用户 |
| 价格带覆盖 | 至少覆盖2个以上主流价格带 |
| 竞争可打 | 不是完全被强品牌垄断，存在中小商家上榜 |
| 产品适配 | 我方供应链能稳定做，且能保证利润或规模 |

### 大爆款词特征

| 适合做大爆款 | 不适合做大爆款 |
| --- | --- |
| 品类核心词 | 极窄定制词 |
| 高频功能词 | 单一小众风格词 |
| 大众场景词 | 极小众人群词 |
| 中高搜索人气词 | 短期热点词 |
| 多竞品成交验证词 | 只有搜索无成交词 |

### 结论模板

> 适合做大爆款的词是【A/B/C】。判断依据是：这些词搜索人气进入TOP20，近30天搜索稳定，支付转化率不低于类目均值，TOP50中有超过【X】个商品承接，说明该词既有足够需求体量，也有成交验证，适合作为主推链接核心关键词。

## 6. 哪些词适合做小爆款？

### 判断目标

判断哪些关键词适合做细分人群、细分场景、细分功能的小爆款。

### 小爆款关键词标准

| 维度 | 明确标准 |
| --- | --- |
| 搜索体量 | 搜索人气排名在TOP20-TOP100，或搜索占比1%-5% |
| 增长潜力 | 近30天搜索增长 ≥ 20% |
| 竞争强度 | 承接商品数少于大盘词，TOP50承接商品少于10个 |
| 需求明确 | 关键词中有人群、场景、功能、风格词 |
| 转化意图 | 词越具体，购买意图越强 |
| 产品差异 | 可以做专门款、专门主图、专门标题 |
| 价格空间 | 可做中价或中高价，不必纯低价竞争 |

### 小爆款词常见类型

| 类型 | 示例 |
| --- | --- |
| 人群词 | 学生、老人、儿童、宝妈 |
| 场景词 | 宿舍、军训、露营、通勤 |
| 功能词 | 防滑、防水、耐脏、显白 |
| 风格词 | 奶油风、复古风、轻奢 |
| 组合词 | 学生宿舍、儿童安全、防滑家用 |

### 结论模板

> 适合做小爆款的词是【A/B/C】。这些词搜索体量不是最大，但近30天增长超过【X%】，且需求非常明确，TOP50中承接商品少于【X】个，说明竞争没有大词激烈，适合通过细分人群、细分场景或功能升级做小爆款。

## 7. 哪些词适合做利润款或动销款？

这里要分开判断。

---

# 7.1 利润款关键词标准

### 判断目标

找出愿意支付更高价格的人群词、品质词、风格词、品牌词、专业词。

### 明确判断标准

| 维度 | 标准 |
| --- | --- |
| 价格带 | 对应商品集中在中高价/高价带 |
| 客单价 | 该词对应商品客单价高于类目均值 ≥ 20% |
| GMV贡献 | GMV占比高于销量占比 |
| 需求类型 | 品质信任、风格审美、身份品牌、专业功能、人群安全 |
| 评价关注 | 用户评价中关注材质、品质、安全、效果、设计 |
| 竞争方式 | 不是单纯比低价 |
| 毛利空间 | 我方核算毛利率能达到目标线 |

### 利润款词类型

| 类型 | 示例 |
| --- | --- |
| 品质词 | A类、无味、真材质、环保、母婴级 |
| 风格词 | 高级感、轻奢、设计感、复古 |
| 专业词 | 医用级、专业、防晒指数、抗菌 |
| 品牌词 | 正品、官方、品牌同款 |
| 礼品词 | 礼盒、送人、高端 |

### 结论模板

> 适合做利润款的词是【A/B/C】。判断依据是：这些词对应商品客单价高于类目均值【X%】，GMV占比高于销量占比，用户评价中高频关注品质、材质、安全或审美，说明用户愿意为价值付费，不适合用低价打法，而适合做中高价利润款。

---

# 7.2 动销款关键词标准

### 判断目标

找出适合做基础销量、店铺权重、长期稳定成交的关键词。

### 明确判断标准

| 维度 | 标准 |
| --- | --- |
| 搜索稳定 | 近30天搜索波动不超过 ±15% |
| 需求常青 | 非节日、非短期热点、非强季节词 |
| 竞争中等 | 有成交，但不是极强红海词 |
| 产品标准化 | 供应链稳定，退货风险低 |
| 价格适中 | 低价/中价均可，重点是成交稳定 |
| 转化稳定 | 转化率不低于类目均值 |
| 维护成本低 | 不需要频繁上新或复杂客服解释 |

### 动销款词类型

| 类型 | 示例 |
| --- | --- |
| 基础品类词 | 桌布、沙发垫、防晒衣 |
| 常规属性词 | 加厚、透明、四季、家用 |
| 稳定功能词 | 防水、防滑、耐脏 |
| 大众场景词 | 家用、客厅、卧室 |

### 结论模板

> 适合做动销款的词是【A/B/C】。判断依据是：这些词搜索稳定，近30天波动不超过【X%】，需求不依赖短期热点，转化率不低于类目均值，产品标准化程度高，适合做长期稳定成交款。

---

## 8. 标题、主图、产品开发分别怎么调整？

### 判断目标

把关键词分析转成实际动作，而不是停留在报告结论。

---

# 8.1 标题怎么调整？

### 判断标准

| 关键词类型 | 标题动作 |
| --- | --- |
| 搜索人气TOP20核心词 | 必须进入标题核心位置 |
| 高增长词 | 进入标题测试位置 |
| 高转化长尾词 | 进入标题长尾组合 |
| 人群词 | 用于做人群标题 |
| 场景词 | 用于场景化标题 |
| 功能词 | 用于卖点标题 |
| 风格词 | 用于风格款标题 |
| 品牌词 | 只有授权或可用时才使用 |

### 标题优先级

| 优先级 | 词类型 |
| --- | --- |
| 第一优先级 | 品类核心词 |
| 第二优先级 | 高搜索功能词 / 场景词 |
| 第三优先级 | 高转化人群词 |
| 第四优先级 | 风格词 / 属性词 |
| 第五优先级 | 长尾补充词 |

### 标题调整标准

| 情况 | 动作 |
| --- | --- |
| 有搜索词，标题没承接 | 加入标题 |
| 标题有词，但无成交 | 换成高转化词 |
| 标题词太泛 | 增加人群/场景/功能长尾词 |
| 高增长词未覆盖 | 做标题AB测试 |
| 标题堆词严重 | 保留核心词，删除弱需求词 |

---

# 8.2 主图怎么调整？

### 判断标准

| 关键词类型 | 主图动作 |
| --- | --- |
| 功能词占比高 | 主图做功能演示 |
| 场景词占比高 | 主图做场景代入 |
| 人群词占比高 | 主图出现目标人群或人群符号 |
| 风格词占比高 | 主图强化审美氛围 |
| 品质词占比高 | 主图展示材质、检测、细节 |
| 效果词占比高 | 主图做前后对比 |
| 价格词占比高 | 主图突出到手价/组合/赠品 |

### 主图调整标准

| 数据表现 | 动作 |
| --- | --- |
| 关键词搜索高，但点击率低 | 主图没有承接搜索意图，重做第一点击理由 |
| 功能词搜索高，但主图没演示 | 增加功能测试画面 |
| 场景词搜索高，但主图泛化 | 换成具体场景图 |
| 人群词搜索高，但主图没有人群感 | 做人群专属主图 |
| 风格词搜索高，但主图普通 | 做风格化视觉 |
| 品质词搜索高，但主图无证明 | 加材质细节/检测背书 |

---

# 8.3 产品开发怎么调整？

### 判断标准

| 关键词结果 | 产品动作 |
| --- | --- |
| 高搜索 + 高增长 + 无自家产品承接 | 新品开发 |
| 高搜索 + 有产品 + 表达弱 | 不开发，先优化标题/主图 |
| 高频差评 + 功能词增长 | 产品升级 |
| 人群词增长 + 现有产品泛化 | 开发人群专属款 |
| 场景词增长 + 现有产品不适配 | 开发场景款 |
| 风格词增长 + 类目主图同质化 | 开发风格款 |
| 高价品质词增长 | 开发利润款 |
| 基础词稳定 | 做动销款或基础款 |

### 产品开发优先级

| 优先级 | 标准 |
| --- | --- |
| 最高优先级 | 搜索占比≥10%，增长≥20%，自家无产品承接 |
| 高优先级 | 评价痛点占比≥10%，且可通过产品升级解决 |
| 中优先级 | 细分词增长快，但搜索体量中等 |
| 低优先级 | 搜索小、增长弱、竞争强 |
| 不开发 | 只有热度，无成交验证，无供应链优势 |

# 最终总表：关键词8个结论的判断标准

| 输出结论 | 核心判断标准 | 最终动作 |
| --- | --- | --- |
| 当前类目最大需求 | 搜索人气占比≥25%，TOP20中出现≥5个，TOP50商品承接≥30% | 做主推方向 |
| 增长最快需求 | 近30天搜索增长≥30%，或3个以上相关词增长≥20% | 做趋势测试 |
| 已有产品承接 | 标题、主图、详情页、SKU、成交反馈中至少4项匹配 | 放大表达或继续优化 |
| 未承接需求 | 有搜索/增长，但自家无SKU、无标题、无主图表达 | 新品开发或链接改版 |
| 大爆款词 | TOP20核心词，搜索占比≥5%，转化不低于均值，TOP50承接≥10个 | 做主推链接核心词 |
| 小爆款词 | TOP20-TOP100，搜索占比1%-5%，增长≥20%，承接商品少于10个 | 做细分款 |
| 利润款词 | 客单价高于均值≥20%，GMV占比高于销量占比，用户关注品质/风格/品牌 | 做中高价利润款 |
| 动销款词 | 搜索波动≤±15%，转化不低于均值，标准化强、维护成本低 | 做稳定动销款 |
| 标题调整 | 高搜索词、高增长词、高转化词未承接 | 标题优化 |
| 主图调整 | 高需求词有搜索但点击低 | 重做第一点击理由 |
| 产品开发 | 高搜索+高增长+无承接，或痛点高频且可解决 | 新品/升级立项 |

一句话标准：

**关键词分析不是为了找词，而是为了判断：哪些需求最大、哪些需求在涨、哪些我们已经承接、哪些还没承接、哪些能做爆款、哪些能做利润款、最后标题、主图、产品怎么改。**
