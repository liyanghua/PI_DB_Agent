# 价格带市场结构判断标准

## 一、先定义价格带

价格带不要凭感觉分，建议按行业前300商品的**券后价/成交价/客单价**分层。

| 价格带 | 定义方式 | 用途 |
| --- | --- | --- |
| 低价带 | 价格低于行业中位价的70% | 判断引流款、价格敏感人群 |
| 中低价带 | 行业中位价70%-90% | 判断高性价比切入 |
| 中价带 | 行业中位价90%-120% | 判断主流成交价格 |
| 中高价带 | 行业中位价120%-160% | 判断品质升级款 |
| 高价带 | 高于行业中位价160% | 判断利润款、品牌款、品质款 |

如果类目价格差距很大，也可以直接按成交价分区，例如：

| 价格带 | 示例 |
| --- | --- |
| 低价带 | 0-49元 |
| 中低价带 | 50-99元 |
| 中价带 | 100-199元 |
| 中高价带 | 200-399元 |
| 高价带 | 400元以上 |

---

## 二、价格带必须统计的字段

| 字段 | 用来判断什么 |
| --- | --- |
| 商品数量占比 | 判断供给集中在哪个价格带 |
| 销量/支付买家数占比 | 判断消费者主要在哪个价格带成交 |
| GMV占比 | 判断金额贡献和利润机会 |
| 平均客单价 | 判断价格带价值 |
| 竞品数量 | 判断竞争强度 |
| 头部品牌数量 | 判断是否被强品牌垄断 |
| 小店/C店数量 | 判断普通商家是否有机会 |
| 毛利率 | 判断我方能不能赚钱 |
| 搜索词价格意图 | 判断用户有没有价格带需求 |
| 关键词增长 | 判断该价格带是否正在变热 |

# 三、价格带判断总表

| 市场结构 | 明确判断标准 | 策略 | 落地动作 |
| --- | --- | --- | --- |
| 市场都低价 | 低价/中低价商品数量占比≥60%，支付买家数占比≥60%，GMV占比低于销量占比；高价带商品数占比＜15% | 可以做高价品质差异，但前提是产品和视觉能支撑 | 做材质升级、功能升级、品牌背书、品质证明，不要直接卷低价 |
| 市场都高价 | 中高价/高价商品数量占比≥60%，GMV占比≥60%，但支付买家数占比不一定高；低价带商品数占比＜15% | 可以做中低价高性价比切入 | 做“同功能更低价”“同品质更划算”“基础款平替” |
| 高中低都有 | 每个价格带商品数量占比都≥15%，且至少2个价格带支付买家数占比≥20% | 找竞争最弱、利润能承接的价格带 | 按产品角色布局：低价引流、中价主推、高价利润 |
| 某价格带商品少但搜索需求高 | 该价格带商品数量占比＜15%，但相关搜索词占比≥10%，或支付买家数占比≥20% | 优先测试 | 小批量上新，先测点击率、收藏加购、转化 |
| 某价格带销量高但毛利低 | 支付买家数占比≥30%，但GMV占比低于销量占比，且毛利率低于目标毛利线 | 适合引流，不适合利润款 | 做流量款、活动款、店铺入口款，不重投利润目标 |
| 某价格带GMV高但竞品弱 | GMV占比≥30%，但头部品牌少，小店/C店占比≥30%，竞品视觉/评价/产品力弱 | 优先开发主推款 | 做主推爆款或中高价利润爆款 |
| 某价格带商品多但销量低 | 商品数量占比≥30%，但支付买家数占比＜20% | 供给过剩，谨慎进入 | 除非有强差异，否则不优先开发 |
| 某价格带商品少、销量少、搜索低 | 商品数占比＜10%，支付买家数占比＜10%，搜索需求弱 | 暂不进入 | 只观察，不开发 |
| 某价格带商品少、GMV高 | 商品数占比＜15%，GMV占比≥25% | 高价值稀缺机会 | 核算供应链和毛利后优先评估 |
| 某价格带销量稳定、波动小 | 近30天成交波动≤±15%，转化率不低于行业均值 | 适合做动销款 | 做基础SKU、稳定成交款 |

# 四、逐条拆解判断标准

## 1. 市场都低价

### 判断标准

满足以下 3 条以上，可以判断为“市场低价化严重”：

| 判断项 | 参考值 |
| --- | --- |
| 低价/中低价商品数量占比 | ≥60% |
| 低价/中低价支付买家数占比 | ≥60% |
| 低价/中低价GMV占比 | ＜支付买家数占比 |
| 高价带商品数量占比 | ＜15% |
| 高价带支付买家数占比 | ＜10% |
| 用户评价中“便宜/划算/性价比”占比 | ≥15% |
| 主图高频打价格、活动、赠品 | TOP50中≥20个 |

### 说明

这说明用户当前大量集中在低价成交，低价带是主流流量池。  
但这不代表只能做低价，也可能说明高价供给不足。

### 策略

可以做高价品质差异，但必须满足以下条件：

| 条件 | 标准 |
| --- | --- |
| 产品有升级点 | 材质、功能、工艺、安全、设计至少1项明显优于低价款 |
| 用户有痛点 | 差评/问大家中品质相关问题占比≥10% |
| 视觉能证明价值 | 主图和详情页能讲清楚为什么贵 |
| 毛利能支撑 | 毛利率达到目标线，例如≥40%或≥50% |
| 有高价小样本成交 | 高价竞品虽然少，但至少有商品稳定成交 |

### 落地动作

| 动作 | 示例 |
| --- | --- |
| 产品升级 | 普通材质 → A类/无味/抗菌/更厚 |
| 视觉升级 | 从价格主图 → 材质细节/品质场景 |
| 详情页升级 | 加检测报告、对比图、使用寿命说明 |
| 价格策略 | 不直接做最高价，先做中高价试水 |
| SKU策略 | 保留低价引流SKU，增加高价值主推SKU |

---

## 2. 市场都高价

### 判断标准

满足以下 3 条以上，可以判断为“市场高价化”：

| 判断项 | 参考值 |
| --- | --- |
| 中高价/高价商品数量占比 | ≥60% |
| 中高价/高价GMV占比 | ≥60% |
| 低价商品数量占比 | ＜15% |
| 低价支付买家数占比 | ＜15% |
| 头部品牌集中度 | TOP10 GMV占比≥40% |
| 用户评价中“品质/品牌/高级感/专业”占比 | ≥15% |
| 高价商品转化率不低于行业均值 | 成立 |

### 说明

说明这个市场用户有支付能力，或者当前被高价品牌占据。  
如果低价供给少，就有“高性价比平替”的机会。

### 策略

可以做中低价高性价比切入。

### 落地动作

| 动作 | 示例 |
| --- | --- |
| 产品策略 | 保留核心功能，减少非必要配置 |
| 价格策略 | 比主流高价低15%-30% |
| 主图策略 | 打“同功能更划算”“高配平替” |
| SKU策略 | 做基础款/升级款/豪华款梯度 |
| 竞品策略 | 对标高价竞品，突出价格优势 |

### 注意

不能只便宜。必须让用户感觉：

**核心价值差不多，但价格更友好。**

---

## 3. 高中低都有

### 判断标准

满足以下条件，可判断为“价格带结构完整”：

| 判断项 | 参考值 |
| --- | --- |
| 低价、中价、高价均有商品 | 每个价格带商品数占比≥15% |
| 至少2个价格带有成交 | 支付买家数占比≥20% |
| 至少2个价格带有GMV贡献 | GMV占比≥20% |
| 不同价格带主卖点不同 | 低价打价格，中价打实用，高价打品质 |
| 不同价格带人群不同 | 价格人群、功能人群、品质人群分层明显 |

### 策略

找竞争最弱、利润能承接的价格带。

### 选择标准

| 判断项 | 优先选择 |
| --- | --- |
| 商品数少 | 供给压力小 |
| 支付买家数高 | 有成交需求 |
| GMV高 | 有金额空间 |
| 毛利能做 | 可持续 |
| 竞品弱 | 容易突破 |
| 搜索词增长 | 有趋势 |
| 小店能上榜 | 门槛不高 |

### 落地动作

| 价格带 | 产品角色 |
| --- | --- |
| 低价带 | 流量款 |
| 中价带 | 主推爆款 |
| 中高价带 | 升级款 |
| 高价带 | 利润款 |
| 空白带 | 测试机会款 |

---

## 4. 某价格带商品少但搜索需求高

### 判断标准

满足以下 2 条以上，即为“价格带空白机会”：

| 判断项 | 参考值 |
| --- | --- |
| 该价格带商品数量占比 | ＜15% |
| 相关关键词搜索人气占比 | ≥10% |
| 相关关键词近30天增长 | ≥20% |
| 该价格带支付买家数占比 | ≥20% |
| 该价格带GMV占比 | ≥20% |
| TOP50中承接商品 | ＜5个 |
| 问大家/评价中用户提到该价格诉求 | ≥10% |

### 策略

优先测试，不要一开始重仓。

### 落地动作

| 动作 | 标准 |
| --- | --- |
| 小批量上新 | 先开发1-3个SKU |
| 测点击 | CTR不低于行业均值，最好≥行业均值1.2倍 |
| 测收藏加购 | 收藏加购率≥行业均值 |
| 测转化 | CVR不低于行业均值 |
| 测价格接受度 | 咨询价格/收藏不买/跳失率 |
| 测7-14天 | 先看基础数据，再决定放量 |

---

## 5. 某价格带销量高但毛利低

### 判断标准

满足以下 3 条以上：

| 判断项 | 参考值 |
| --- | --- |
| 支付买家数占比 | ≥30% |
| GMV占比低于支付买家数占比 | 成立 |
| 客单价低于行业均值 | ≥20% |
| 毛利率低于目标毛利线 | 例如＜30%或低于店铺要求 |
| 价格竞争激烈 | TOP50中大量商品打折/券后低价 |
| 用户关键词偏价格 | “便宜/划算/平替/低价”占比≥15% |
| 竞品视觉普遍打价格利益 | TOP50中≥20个 |

### 策略

适合引流，不适合利润款。

### 落地动作

| 动作 | 说明 |
| --- | --- |
| 做流量款 | 用来拉访客、拉搜索权重 |
| 控制投放 | 不用高成本投放硬推利润 |
| 做组合带动 | 用低价款带动中高价SKU |
| 做关联销售 | 详情页推荐利润款 |
| 控制成本 | 供应链必须极致控制 |
| 不做复杂服务 | 避免低毛利高售后 |

### 判断结论模板

> 该价格带支付买家数占比达到38%，但GMV占比只有24%，且毛利率低于目标线，说明该价格带有流量但利润弱，适合作为引流款，不适合作为利润款。

---

## 6. 某价格带GMV高但竞品弱

### 判断标准

满足以下 3 条以上：

| 判断项 | 参考值 |
| --- | --- |
| 该价格带GMV占比 | ≥30% |
| 该价格带支付买家数占比 | ≥20% |
| TOP10品牌集中度 | ＜40% |
| 小店/C店占比 | ≥30% |
| 竞品主图同质化 | ≥70% |
| 竞品差评集中 | 某痛点占比≥10% |
| 竞品详情页表达弱 | 前3屏没有讲清核心卖点 |
| 竞品客服/服务弱 | 高频顾虑回答不清 |
| 我方毛利能承接 | 毛利率达到目标线 |

### 策略

优先开发主推款。

### 落地动作

| 方向 | 动作 |
| --- | --- |
| 产品 | 针对竞品差评做升级 |
| 视觉 | 针对竞品同质化做差异主图 |
| 价格 | 不一定最低，但要价值感更强 |
| SKU | 做主推SKU + 利润SKU |
| 详情页 | 前3屏解决用户最大顾虑 |
| 客服 | 针对高频问大家做标准话术 |
| 投放 | 可作为主推款进入测试放量 |

# 五、价格带机会分级

## A级机会：优先开发

满足以下任意 3 条：

| 条件 |
| --- |
| GMV占比≥30% |
| 支付买家数占比≥25% |
| 商品数量占比＜20% |
| 关键词增长≥20% |
| 竞品弱，小店/C店占比≥30% |
| 毛利率达到目标线 |
| 评价痛点占比≥10%且我方能解决 |

动作：

**立项开发 / 主推款规划 / 进入测款。**

---

## B级机会：小批量测试

满足以下任意 3 条：

| 条件 |
| --- |
| 搜索需求占比≥8% |
| 关键词增长≥15% |
| 商品数量占比＜25% |
| 有少量竞品成交 |
| 我方有供应链优势 |
| 毛利基本可做 |
| 主图有差异机会 |

动作：

**小批量上新，7-14天测试点击、加购、转化。**

---

## C级机会：观察

满足以下情况：

| 条件 |
| --- |
| 搜索增长有，但成交未验证 |
| 商品少，但用户需求不明显 |
| GMV不高，销量也不高 |
| 供应链不确定 |
| 毛利不稳定 |

动作：

**只做监控，不马上开发。**

---

## D级机会：不建议进入

满足以下任意 2 条：

| 条件 |
| --- |
| 商品数占比高但销量占比低 |
| 毛利无法承接 |
| 头部品牌垄断严重 |
| 搜索需求低且无增长 |
| 售后风险高 |
| 我方没有明显优势 |

动作：

**暂不进入。**

# 六、价格带策略落地表

| 市场结构 | 判断标准 | 产品角色 | 定价策略 | 主图策略 | 产品动作 | 验证指标 |
| --- | --- | --- | --- | --- | --- | --- |
| 市场都低价 | 低价商品数/买家数≥60% | 中高价品质款 | 比低价款高，但必须有价值证明 | 材质、功能、品质证明 | 材质升级、功能升级 | CTR≥行业均值，CVR不低于行业均值 |
| 市场都高价 | 高价商品数/GMV≥60% | 高性价比款 | 比主流高价低15%-30% | 平替、同功能更划算 | 做基础款/精简款 | 点击率提升，转化不低于均值 |

| GMV高但竞品弱 | GMV≥30%，小店/C店≥30% | 主推款/利润款 | 中价或中高价 | 差异化卖点 | 产品升级 | GMV、转化、利润 |

# 七、最终判断公式

价格带机会可以用这个公式判断：

```text
价格带机会 = 需求强度 × 成交验证 × 竞争强度 × 毛利空间 × 我方优势

```

具体解释：

| 因素 | 看什么 |
| --- | --- |
| 需求强度 | 搜索人气、关键词增长、问大家需求 |
| 成交验证 | 支付买家数、GMV、TOP商品成交 |
| 竞争强度 | 商品数、头部品牌、小店/C店、视觉同质化 |
| 毛利空间 | 客单价、成本、毛利率 |
| 我方优势 | 供应链、视觉、价格、品牌、客服、资源 |

---

# 八、最终总结

价格带分析最终不是回答“卖多少钱”，而是回答：

1.  **哪个价格带有流量？**
    
2.  **哪个价格带有GMV？**
    
3.  **哪个价格带竞争弱？**
    
4.  **哪个价格带能赚钱？**
    
5.  **哪个价格带适合我们切入？**
    

最核心的判断标准是：

| 判断问题 | 看什么 |
| --- | --- |
| 有没有需求 | 搜索占比、关键词增长 |
| 有没有成交 | 支付买家数占比 |
| 有没有钱赚 | GMV占比、毛利率 |
| 竞争强不强 | 商品数量、头部集中度、小店占比 |
| 我们能不能打 | 产品、视觉、价格、供应链优势 |

一句话：

**低价市场看能不能做品质升级，高价市场看能不能做高性价比切入，结构完整市场看哪个价格带竞争最弱，空白价格带看搜索和成交是否成立，销量高毛利低适合引流，GMV高竞品弱适合做主推。**
