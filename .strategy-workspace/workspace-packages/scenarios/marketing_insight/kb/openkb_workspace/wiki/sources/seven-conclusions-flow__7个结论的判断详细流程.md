# 2.5 分析结论模板：7个结论的判断依据

## 1. 当前行业热卖产品分为哪几类？

### 判断依据

看行业前300商品中的这些字段：

| 字段 | 用来判断什么 |
| --- | --- |
| 产品类型 | 行业主要卖哪些品类/款式 |
| 款式名称 / SKU名称 | 判断商品形态 |
| 材质 | 判断材质型产品分类 |
| 功能 | 判断功能型产品分类 |
| 风格 | 判断风格型产品分类 |
| 使用场景 | 判断场景型产品分类 |
| 主图 | 判断视觉和场景表达 |
| 标题关键词 | 判断产品核心卖点和需求入口 |

### 具体判断方法

把行业前300商品按“核心购买理由”归类，而不是只按商品名称归类。

例如沙发垫不能只分成“沙发垫、沙发巾、沙发套”，还要分成：

| 分类方式 | 示例 |
| --- | --- |
| 按材质分 | 冰丝款、半边绒款、羊羔绒款、雪尼尔款 |
| 按风格分 | 奶油风、轻奢风、复古风、简约风 |
| 按功能分 | 防滑款、防水款、A类面料款、可机洗款 |
| 按场景分 | 客厅款、宠物家庭款、母婴家庭款、出租房款 |
| 按形态分 | 普通沙发垫、异形沙发垫、沙发巾、沙发笠 |

### 可执行标准

某一类产品满足以下条件之一，就可以判定为“行业热卖类型”：

| 判断标准 | 说明 |
| --- | --- |
| 数量占比 ≥ 10% | 行业前300中至少出现30个 |
| 销量/支付买家数占比 ≥ 10% | 不只是数量多，而且真实卖得动 |
| TOP50中出现 ≥ 5个 | 说明进入头部竞争区 |
| 多个头部店铺都有布局 | 说明不是个别店铺偶然爆 |
| 搜索关键词中有对应需求词 | 说明用户有主动需求 |

### 输出格式（反向约束提示词不能省略）

不要写：

“行业主要有多种产品类型。”

要写：

“当前行业热卖产品主要分为4类：A类、B类、C类、D类。其中A类在前300中出现86个，占比28.7%，集中在中价位，主要卖点是××；B类出现52个，占比17.3%，增长较快，主要原因是××。”

## 2. 每类产品的市场占比大概是多少？

### 判断依据

看这些字段：

| 字段 | 作用 |
| --- | --- |
| 商品数量 | 判断供给占比 |
| 销量 或支付买家数 | 判断成交占比 |
| GMV 或交易指数 | 判断市场金额占比 |
| 客单价 | 判断高低价贡献 |
| 排名位置 | 判断头部集中度 |

### 具体计算方法

至少算三个占比：

#### 1）数量占比

```text
某类型商品数量 ÷ 行业前300商品总数

```

例如：

行业前300中，半边绒沙发垫有60个。

```text
60 ÷ 300 = 20%

```

结论：

“半边绒款在行业前300中数量占比20%，属于主流供给类型。”

#### 2）销量占比

```text
某类型商品支付买家数 ÷ 前300总支付买家数

```

这个比数量占比更重要，因为数量多不代表卖得好。

#### 3）GMV占比

```text
某类型商品GMV ÷ 前300总GMV

```

GMV占比可以判断这个类型是不是赚钱型市场。

### 判断标准

| 占比结果 | 判断 |
| --- | --- |
| 数量占比高，销量占比高 | 主流热卖类型 |
| 数量占比低，销量占比高 | 高效率机会类型 |
| 数量占比高，销量占比低 | 供给过剩，谨慎进入 |
| 销量占比高，GMV占比低 | 低价走量市场 |
| 销量占比低，GMV占比高 | 高客单利润市场 |
| 数量占比低，GMV占比高 | 高价值稀缺机会 |

### 输出格式

| 产品类型 | 商品数占比 | 销量占比 | GMV占比 | 判断 |
| --- | --- | --- | --- | --- |
| A类 | 28% | 35% | 30% | 主流爆款类型 |
| B类 | 12% | 25% | 18% | 高效率机会类型 |
| C类 | 30% | 15% | 12% | 供给多但效率低 |
| D类 | 8% | 5% | 20% | 高客单利润机会 |

## 3. 行业主流价格带是什么？

### 判断依据

看这些字段：

| 字段 | 作用 |
| --- | --- |
| 券后价 / 到手价 | 真实成交价格 |
| 客单价 | 判断真实支付水平 |
| SKU价格 | 判断主销价格 |
| 销量 / 支付买家数 | 判断哪个价格带买的人最多 |
| GMV | 判断哪个价格带金额最大 |
| 上榜宝贝数量 | 判断供给集中在哪个价格带 |

### 具体判断方法

第一步：先按价格把商品分组。可参考生意参谋6个价格带

例如：

| 价格带 | 区间 |
| --- | --- |
| 低价带 | 0-49元 |
| 中低价带 | 50-99元 |
| 中价带 | 100-199元 |
| 中高价带 | 200-399元 |
| 高价带 | 400元以上 |

第二步：每个价格带算 4 个数据：

| 数据 | 计算方式 |
| --- | --- |
| 商品数量占比 | 该价格带商品数 ÷ 前300商品数 |
| 销量占比 | 该价格带支付买家数 ÷ 总支付买家数 |
| GMV占比 | 该价格带GMV ÷ 总GMV |
| 平均客单价 | 该价格带GMV ÷ 支付买家数 |

第三步：判断主流价格带。

### 判断标准

| 价格带表现 | 结论 |
| --- | --- |
| 商品数占比最高 + 销量占比最高 | 行业主流成交价格带 |
| 销量占比最高，但GMV占比低 | 低价走量价格带 |
| GMV占比最高，但销量占比中等 | 高客单利润价格带 |
| 商品数少，但销量高 | 供给不足的机会价格带 |
| 商品数多，但销量低 | 竞争拥挤、效率差价格带 |

### 输出格式

“行业主流成交价格带是100-199元。判断依据是：该价格带在前300商品中数量占比32%，支付买家数占比38%，GMV占比35%，三项均为最高，说明该价格带既有供给集中，也有成交集中，是当前行业主流价格带。”

## 4. 高价、中价、低价分别卖什么人群？

### 判断依据

看这些字段：

| 字段 | 作用 |
| --- | --- |
| 价格带 | 区分高/中/低价 |
| 主图表达 | 判断人群心智 |
| 标题关键词 | 判断人群需求 |
| 详情页卖点 | 判断价值主张 |
| 评价内容 | 判断用户关注点 |
| 问大家 | 判断购买顾虑 |
| 达摩盘竞品人群 | 判断真实人群画像 |
| SKU销售占比 | 判断哪个人群真实购买 |

### 具体判断方法

不要直接用“高价=高端人群、低价=低端人群”这种粗暴判断。

要从“价格 + 需求 + 表达 + 评价”四个维度交叉判断。

| 价格带 | 看什么 | 判断人群 |
| --- | --- | --- |
| 低价带 | 标题是否有便宜、宿舍、基础、简易；评价是否关注划算 | 价格敏感人群、学生党、刚需人群 |
| 中价带 | 是否强调实用、耐用、颜值、性价比 | 大众家庭、主流消费人群 |
| 高价带 | 是否强调材质、设计、品牌、安全、专业、风格 | 品质人群、审美人群、专业人群、送礼人群 |

### 判断标准参考详细文档：[《8大人群判断标准》](https://alidocs.dingtalk.com/i/nodes/9bN7RYPWdMYPzBP2ikQY9xdYVZd1wyK0)

| 8大人群 | 证据组合 | 可判断人群 | 常见价格带 | 判断依据 | 落地动作 |
| --- | --- | --- | --- | --- | --- |
| 价格人群 | 低价 + “便宜/划算/平替/宿舍/临时用/性价比/低价”关键词 | 价格敏感人群 | 低价 / 中低价 | 用户核心决策是“少花钱买到能用的” | 做低价引流款、基础款、活动款 |
| 功能人群 | 中高价 + “防滑/防水/防晒/抗菌/遮光/耐磨/省心/易清洗”关键词 | 功能刚需人群 | 中价 / 中高价 | 用户不是单纯买产品，而是买一个明确功能解决方案 | 主图突出功能演示，详情页证明功能效果 |
| 效果人群 | 中高价 + “显白/显瘦/降温/变好看/防晒效果/遮瑕/改善/前后对比”关键词 | 结果导向人群 | 中价 / 中高价 | 用户关心买完后的结果变化 | 用对比图、效果图、真实反馈证明 |
| 场景人群 | 低中高价均可能 + “宿舍/客厅/卧室/通勤/露营/军训/母婴/宠物/出租房”关键词 | 场景适配人群 | 低价 / 中价 / 中高价 | 用户不是泛买，而是在某个具体场景下买 | 主图直接场景化，标题加入场景词 |
| 审美人群 | 中高价 + “高级感/轻奢/奶油风/复古风/多巴胺/ins风/简约/设计感”关键词 | 风格审美人群 | 中价 / 中高价 / 高价 | 用户买的是“好看、搭配、风格统一” | 主图强化风格氛围，详情页做搭配场景 |
| 品质信任人群 | 高价 + “A类/安全/无味/真材质/环保/母婴级/检测报告/不掉色/不掉毛”关键词 | 安全品质人群 | 中高价 / 高价 | 用户担心踩雷，需要安全感和确定性 | 强化材质证明、检测报告、评价背书 |
| 身份品牌人群 | 高价 + “品牌/正品/礼盒/送人/高端/专业/大牌同款/官方”关键词 | 品牌身份人群 | 高价 / 超高价 | 用户买的是身份、体面、品牌背书 | 强化品牌、包装、送礼、专业背书 |
| 情绪悦己人群 | 中高价 + “治愈/可爱/氛围感/少女心/松弛感/精致/变美/喜欢”关键词 | 情绪消费人群 | 中价 / 中高价 | 用户不一定刚需，但被情绪、审美、生活方式打动 | 做内容种草、氛围主图、情绪化文案 |

## 5. 行业爆款共性是什么？

### 判断依据

看行业TOP50或TOP100，而不是只看前300平均值。

重点看这些字段：

| 字段 | 作用 |
| --- | --- |
| TOP排名 | 判断爆款样本 |
| 销量 / 支付买家数 | 判断是否真爆 |
| 增长率 | 判断是否趋势爆款 |
| 材质 | 判断产品共性 |
| 功能 | 判断购买理由 |
| 风格 | 判断审美趋势 |
| 场景 | 判断使用环境 |
| 主图 | 判断点击表达 |
| 标题关键词 | 判断搜索入口 |
| 评价好评点 | 判断用户认可原因 |

### 具体判断方法

把TOP50商品拆成 8 个共性：

| 共性类型 | 判断问题 |
| --- | --- |
| 材质共性 | TOP50里高频材质是什么？ |
| 功能共性 | 是否都在解决同一个痛点？ |
| 风格共性 | 是否集中在某种审美风格？ |
| 价格共性 | 是否集中在某个价格带？ |
| 场景共性 | 是否集中表达某个使用场景？ |
| 人群共性 | 是否集中服务某类人群？ |
| 视觉共性 | 主图是否有共同表达方式？ |
| 卖点共性 | 第一卖点是否高度一致？ |

### 判断标准

| 出现频率 | 结论 |
| --- | --- |
| TOP50中出现 ≥ 30个 | 强爆款共性 |
| TOP50中出现 15-29个 | 中等爆款共性 |
| TOP50中出现 5-14个，但增长快 | 趋势型共性 |
| TOP50中少，但在飙升商品中多 | 新趋势共性 |
| 只在个别商品出现 | 暂不判定为行业共性 |

### 案例表达

沙发垫案例可以这样写：

“行业爆款共性集中在三个方面：第一，材质上半边绒属于四季热销面料；第二，设计元素上黑白条纹属于高频爆款元素；第三，风格上现代轻奢适配大部分客厅装修风格。因此，半边绒 + 黑白条纹 + 现代轻奢，可以作为当前沙发垫类目的一个爆款基因组合。”

这个判断来自沙发垫文档中对款式起来原因的示例：从面料、设计元素、款式风格三个方面总结，例如半边绒、黑白条纹、现代轻奢。

## 6. 行业差异机会在哪里？

### 判断依据

差异机会不是凭空想出来的，而是从 5 个地方找：

| 来源 | 判断依据 |
| --- | --- |
| 热卖产品同质化 | 大量商品材质、风格、主图、卖点相似 |
| 关键词需求未被满足 | 有搜索增长，但商品少 |
| 评价痛点未被解决 | 用户反复抱怨，竞品没解决 |
| 价格带空白 | 有需求，但某价格带供给少 |
| 跨平台趋势 | 小红书/抖音火了，淘宝还少 |

### 具体判断方法

#### 机会1：产品差异机会

判断依据：

```text
用户有需求 + 行业产品同质化 + 竞品没有明显升级

```

动作：

做材质升级、功能升级、结构升级。

#### 机会2：人群差异机会

判断依据：

```text
关键词中出现明确人群词 + 竞品主图/详情页没有针对这个人群表达

```

动作：

做“极致人群”定位。

例如：学生党、宝妈、老人、宠物家庭、精致女生。

#### 机会3：场景差异机会

判断依据：

```text
场景词有搜索量/增长 + 竞品只做泛场景表达

```

动作：

主图和详情页改成具体场景。

例如：宿舍、出租屋、客厅、儿童房、露营、通勤。

#### 机会4：价格带差异机会

判断依据：

```text
某价格带商品少 + 搜索需求存在 + 成交效率不错

```

动作：

布局该价格带产品。

#### 机会5：视觉差异机会

判断依据：

```text
TOP竞品主图表达高度相似 + 用户需求没有被一眼表达

```

动作：

做不同背景、构图、场景、第一卖点表达。

#### 机会6：跨类目借鉴机会

判断依据：

```text
其他相关类目元素火 + 本类目还没大量出现

```

动作：

把外部爆点迁移到本类目。

### 输出格式

| 差异机会 | 判断依据 | 具体动作 |
| --- | --- | --- |
| 材质升级 | TOP竞品集中使用普通材质，评价中反复出现闷/滑/掉毛 | 开发升级材质款 |
| 人群细分 | 关键词中“学生/儿童/老人”增长，但主图没有人群表达 | 做专属人群定位 |
| 场景细分 | “宿舍/客厅/露营”搜索增长，竞品表达泛 | 做场景化主图 |
| 价格带机会 | 中价带供给少但成交效率高 | 布局中价主推款 |
| 视觉机会 | 行业主图高度同质化 | 做差异化视觉锤 |

## 7. 哪些产品涨得快，原因是什么？

### 判断依据

看这些字段：

| 字段 | 作用 |
| --- | --- |
| 排名变化 | 判断是否快速上升 |
| 近7天/近30天销量增长 | 判断成交增长 |
| 搜索人气增长 | 判断需求增长 |
| 关键词增长 | 判断流量入口变化 |
| 店铺上新节奏 | 判断是否新趋势布局 |
| 主图/卖点变化 | 判断表达是否带来增长 |
| 跨平台热度 | 判断外部趋势带动 |
| 是否小店/C店 | 判断机会是否容易复制 |

### 具体判断标准

| 增长现象 | 判断 |
| --- | --- |
| 排名提升 ≥ 20名 | 高增速趋势新品 |
| 近7天搜索涨幅明显 | 需求正在上升 |
| 近30天支付买家数持续增长 | 成交趋势成立 |
| 小店/C店快速上榜 | 产品机会大于品牌优势 |
| 多个平台同时出现 | 外部趋势正在扩散 |
| 关键词增长 + 产品增长同步 | 真实需求趋势 |
| 只有销量增长，没有关键词增长 | 可能是活动/付费拉动，要谨慎 |

### 原因分析维度

涨得快的原因不能只写“趋势好”，必须拆成 6 类：

| 原因类型 | 判断依据 | 示例 |
| --- | --- | --- |
| 需求趋势 | 搜索词增长、下拉词出现 | 多巴胺、腊肠狗、军训防晒 |
| 材质趋势 | 新材质高频出现 | 冰丝、半边绒、羊羔绒 |
| 风格趋势 | 主图风格和内容平台一致 | 奶油风、轻奢风、复古风 |
| 功能升级 | 解决高频痛点 | 防滑、防水、无味、A类 |
| 价格优势 | 同类低价或高性价比 | 比竞品低10%-20% |
| 视觉优势 | 主图点击理由更强 | 一眼看懂场景/效果/差异 |

### 输出格式

| 涨得快的产品 | 增长证据 | 原因判断 | 是否值得跟进 | 跟进方式 |
| --- | --- | --- | --- | --- |
| A款 | 排名提升35名，搜索词增长，竞品少 | 新风格趋势 + 供给不足 | 高 | 快速找款上架 |
| B款 | 销量增长但无搜索增长 | 可能是活动拉动 | 中 | 观察7天 |
| C款 | 小店上榜，主图差但销量高 | 产品需求强，视觉可升级 | 高 | 做更强视觉版本 |

# 最终可执行判断表

你可以把 7 个结论统一做成这张表：

| 结论问题 | 核心判断依据 | 计算/判断方法 | 最终动作 |
| --- | --- | --- | --- |
| 热卖产品分几类 | 产品类型、材质、功能、风格、场景 | 前300分类统计 | 确定主流产品方向 |
| 每类占比多少 | 商品数、销量、GMV | 数量占比、销量占比、GMV占比 | 判断主流/小众/机会 |
| 主流价格带 | 券后价、客单价、销量、GMV | 按价格区间统计 | 确定定价区间 |
| 高中低价卖谁 | 价格+关键词+主图+评价+达摩盘 | 交叉判断人群 | 做人群定位 |
| 爆款共性 | TOP50材质、功能、风格、主图、卖点 | 高频元素统计 | 提炼爆款基因 |
| 差异机会 | 关键词缺口、评价痛点、价格空白、竞品弱点 | 找对手没有做/少/弱 | 做产品创新 |
| 涨得快的产品 | 排名变化、销量增长、搜索增长、跨平台热度 | 增长归因 | 快速跟进或测试 |

一句话：

**这 7 个结论的判断依据，就是把行业前300商品、关键词、评价问大家、价格带、竞品、跨平台趋势这六类数据交叉验证，最后判断什么是主流、什么是趋势、什么是机会、什么能落地。**
